OWNER VIEW (35/65 SPLIT)
  Left (35%): Item inventory list
  Right (65%): Form with OFFER + REQUEST sections
    - OFFER: What shop gives (quantity input + stock display)
    - REQUEST: What shop wants (quantity + item type inputs)
    - SUMMARY: Arrow-based transaction display
    - ADD TRADE: Button with validation
    - ACTIVE TRADES: List with delete buttons
    - FOOTER: SAVE ALL + CANCEL

  Right (40%): Details panel

    - Trade selector (radio button replacement)
    - Affordability check display
    - Action button (ACCEPT TRADE)

VALIDATION:
  - Real-time feedback (green/red text)
  - Disabled state for invalid inputs
  - Clear error messages
  - Prevents impossible trades
