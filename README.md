# Just Another Shop Mod

**Just Another Shop Mod** lets players and server admins turn ordinary storage crates into fully functional, interactive storefronts in Project Zomboid (Build 42).

## Features

- **Create your own store:** Easily register standard crates as shops for your base, town, or trading outposts.
- **Dedicated shop menus:** Simple, distinct interfaces for customers to browse goods and for owners to manage stock.
- **Player and Admin shops:** Supports independent player-run stores as well as system-operated admin shops. _(WIP)_
- **Secure ownership:** Only the designated shop owner or server administrators can access the shop's management settings.
- **Safe trading:** Built-in lock systems prevent multiple players from accessing and exploiting a shop at the exact same time.

## Usage

- To setup a store, simply place an allowed crate (e.g., standard Wood or Metal crates).
- Right-click and navigate to the **JASM Shop** menu to register the container.
- After registration, owners can **Manage Shop** to adjust stock and prices, and customers can use **Open Shop UI** to browse and buy goods.
